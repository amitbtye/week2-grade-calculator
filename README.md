# Student Grade Calculator

## Project Description 
This project calculates grades of students based on marks of 3 subjects.It also provides comments and class statistics.

## Features
Multiple students input
Grade calculation (A-f)
personalized comments
class statistics(average,higest,lowest)
Error handling
File saving

Lists and loops
Functions
Error handling using try-except
